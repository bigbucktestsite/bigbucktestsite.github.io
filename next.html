Next steps for Big Buck Hunter: Restoration

Big Buck Hunter: Restoration took a long time to finish. Ideally, now that it is complete, I would like to see it placed permanently in a bar or smaller restaurant in West Virginia, the type of place I saw the original game in while growing up.

I'm also working on making a small modkit that will either be for sale or DIYable, a small computer system that can be installed in an original Big Buck Hunter arcade machine to make it run Big Buck Hunter: Restoration. It is uncertain if this would be legal to sell since it is technically redistributing code from the original Big Buck Hunter game, once and if these issues are worked out, this website will be updated with the information.

For now, it will be on display at l'Avant Galerie Vossen, in Paris, from January 23rd until early March. After that, it will be on display in the United States.
